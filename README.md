# Multiplication App Node TypeScript

## Correr en Dev

```
npm install
```

```
npm run dev
```

## Ejemplo enviando argv:

```
npx ts-node src/app.ts --base 6 -s -n table-6 -d outputs/table-6-folder
```

donde:

- --base: number base multiplication
- -s : show table
- -n : name de file table
- -d : destination file
